⚙️ Opciones de ingreso

📶 NFC

Lee una etiqueta NFC (ISBN/identificador) y búscalo en tu biblioteca. (Solo Android + Chrome con NFC.)

❔ Qué hace cada método de ingreso
📷 Cámara — foto de portada y contraportada; autorecorte y automedida con el tapete (la forma preferida para libros físicos).
🔢 Ingreso por ISBN — por código de barras: recupera los datos del Fichero local y busca la portada.
📁 Subir ficheros — EPUB/PDF/imágenes/cómic/.zip; varias imágenes del mismo libro → un libro.
🏷️ Datos de esta alta — ISBN / colección / obra / ubicación que viajan con lo que subas.
📲 Subir desde otra app — importa de Adobe Scan / Drive / Files, o abre la cámara del móvil.
⚙️ Opciones de ingreso — supervisado, procesar lo compartido al instante, elegir portada.
📶 NFC — leer una etiqueta (Android + Chrome).
